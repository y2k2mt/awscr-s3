    class Awscr::S3::FileUploader

    • Awscr::S3::FileUploader
    • Reference
    • Object

    Overview

    Uploads a file to S3. If the file is 5MB it is uploaded in a single request. If the file is greater than 5MB it is uploaded in parts.
